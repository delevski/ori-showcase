# Ori Showcase

A polished single-page web app that introduces Ori Delevski and highlights selected public GitHub repositories.

## Features

- Animated glassmorphism UI
- Hero section with leadership / product focus
- Filterable featured repo cards
- Full public repo cloud
- Responsive layout
- Playwright e2e tests
- GitHub Actions CI

## Scripts

```bash
npm install
npm test
npm run serve
```

## Data

The app loads a local snapshot of public GitHub repository metadata from `data/profile.json` and `data/repos.json`.
